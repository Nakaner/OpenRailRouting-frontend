About OpenRailRouting

OpenRailRouting is a routing engine for railway vehicles on the railway network mapped in the OpenStreetMap database. The routing engine is based on GraphHopper, an OpenStreetMap routing engine in Java.

OpenRailRouting is free and open source software under the Apache License version 2. You can get the source code on GitHub. Build instructions for the backend written in Java can be found there.

The development of OpenRailRouting was sponsored by SNCF, the national railway company of France which heavily makes use of OpenStreetMap and its railway data.

OpenRailRouting API

OpenRailRouting (the backend) has an HTTP REST API which is almost identical to the API of GraphHopper. You can find the documentation currently on GitHub only.
